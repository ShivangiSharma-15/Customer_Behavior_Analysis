Overview

End-to-end data analytics project focused on extracting actionable insights from raw data. The workflow covers data ingestion, cleaning, exploratory data analysis (EDA), SQL-based analysis, and visualization through dashboards and reports.

📁 Dataset
Source: [customer_shopping_behavior.csv]
Type: Structured dataset (CSV)
Size: [Rows=3900/Columns=18]
Features include: Customer ID,	Age,	Gender,	Item Purchased,	Category,	Purchase Amount (USD),	Location,	Size,	Color,	Season,	Review Rating,	Subscription Status,	Shipping Type,	Discount Applied,	Promo Code Used,	Previous Purchases,	Payment Method,	Frequency of Purchases

🛠️ Tools & Technologies
Python: Pandas, NumPy, Matplotlib, Seaborn
Database: PostgreSQL 
SQL: Data querying, aggregation, joins
Visualization: Power BI
Reporting: PowerPoint (Gamma)
Environment: Jupyter Notebook 

⚙️ Project Workflow
1. Data Loading
Imported dataset into Python using Pandas
Connected Python to SQL database using SQLAlchemy
2. Data Cleaning
Handled missing values
Removed duplicates
Standardized column names and formats
3. Exploratory Data Analysis (EDA)
Uncovered trends and patterns
Analyzed distributions, correlations, and outliers
Generated visual insights using plots and charts
4. SQL Analysis
Loaded cleaned data into database
Wrote queries to extract business insights:
Revenue analysis
Customer segmentation
Product performance
Used aggregations, GROUP BY, JOINs
5. Dashboard Creation
Built interactive Power BI dashboard
Visualized KPIs, trends, and insights
Enabled filtering for better decision-making
6. Reporting
Created a structured report summarizing findings
Designed a presentation using Gamma for stakeholder communication

📊 Dashboard Highlights
Revenue trends over time
Customer segmentation analysis
Top-performing products
Interactive filters for dynamic insights

📈 Key Results
Identified key revenue drivers and customer behavior patterns
Improved data clarity through preprocessing and visualization
Delivered actionable insights for business decision-making
